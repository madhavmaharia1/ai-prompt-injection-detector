# 🛡️ AI Prompt Injection Detector

A real-time web-based tool that detects prompt injection attacks in LLM inputs using rule-based analysis and risk scoring.

---

## 🔍 Features

* Detects prompt injection and jailbreak attempts
* Flags suspicious instruction overrides
* Assigns a risk score to user input
* Displays real-time analysis results
* Logs recent activity for monitoring

---

## 🧠 Concepts Used

* OWASP LLM Top 10 (LLM01: Prompt Injection)
* MITRE ATLAS
* Rule-based detection (regex patterns)

---

## 🚀 How to Run

```bash
pip install -r requirements.txt
python app.py
```

Then open: http://127.0.0.1:5000

---

## 🛠️ Tech Stack

* Python
